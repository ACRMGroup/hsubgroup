Human Subgroup Assignment

This page allows you to identify the subgroup of a human light or heavy chain. The code which does the actual assignments was kindly supplied by Sophie Deret and it normally features as part of her Subim program.

Your WWW browser must support fill-out forms with the POST method for you to use this facility! Lynx V2.2 doesn't seem to work, but V2.3.7 BETA is fine.
